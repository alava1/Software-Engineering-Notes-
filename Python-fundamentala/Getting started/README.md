# 🐍 What is Python?

Python is a **high-level**, **interpreted**, and **general-purpose** programming language created by **Guido van Rossum** and officially released in **1991**.

Python focuses on readability, simplicity, and productivity, making it one of the best programming languages for beginners and professionals alike.

# ⚙️ Installing Python

1. Download Python from the official Python website.
2. Install Python.
3. During installation, ensure **"Add Python to PATH"** is selected.
4. Verify the installation by running:

```bash
python --version
```

or

```bash
python3 --version
```
